## Installation

Install tokyo with npm

```bash
  cd /tokyo
  npm install
```

Create and fill config.env\
It will work with MongoDB Atlas for DATABASE\
and mailtrap as an EMAIL options

```bash
NODE_ENV=development
PORT=

DATABASE=
DATABASE_PASSWORD=

EMAIL_HOST=
EMAIL_PORT=
EMAIL_USERNAME=
EMAIL_PASSWORD=
```
